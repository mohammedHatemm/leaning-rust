# helloProject

This is a simple Rust project.

## Project Structure

- `Cargo.toml`: Project manifest file.
- `src/main.rs`: Main source code file.

## Commands

To run this project, navigate to the project root directory and execute the following command:

```bash
cargo run
```

## Functions

The main function in `src/main.rs` is:

```rust
fn main() {
    println!("Hello, rust form the other world !");
}
```

This function prints the message "Hello, rust form the other world !" to the console.
